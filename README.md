# SI-PEJALAN - Sistem Informasi Pendataan Kerusakan Jalan

Sistem Informasi berbasis web untuk pendataan kerusakan jalan yang memungkinkan pelaporan, penyimpanan, pemantauan, dan pelaporan prioritas perbaikan kerusakan jalan.

## Fitur Utama

- **Pelaporan Kerusakan**: Form pelaporan dengan input lokasi, tipe kerusakan, tingkat kerusakan, deskripsi, dan foto
- **Pemetaan Interaktif**: Tampilkan laporan pada peta dengan marker clustering
- **Dashboard Monitoring**: Statistik dan grafik untuk monitoring progress perbaikan
- **Manajemen Laporan**: Filter, pencarian, dan pengelolaan laporan kerusakan
- **Multi-role Support**: Support untuk Reporter, Petugas, Admin, dan Manager

## Teknologi yang Digunakan

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **UI Framework**: Bootstrap 5.3
- **Maps**: Leaflet.js dengan OpenStreetMap
- **Charts**: Chart.js untuk visualisasi data
- **Icons**: Font Awesome 6
